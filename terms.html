2. Description of Service

Gold-B's Agent is a self-hosted multi-channel messaging automation platform that runs locally on the user's hardware via Docker containers. The Application provides:

  • Multi-channel message processing and responses (WhatsApp, Telegram, Instagram, Facebook Messenger, SMS, Email)
  • Keyword-based rule matching and template-based replies
  • A visual flow builder for designing complex automation workflows
  • AI-powered response generation via third-party AI providers
  • Knowledge base with document upload (PDF, DOCX, TXT) and AI-powered retrieval (RAG)
  • Knowledge Sources — automated data sync from 10 platforms into AI knowledge agents
  • Optional integrations with Google (Sheets, Drive, Gmail, Calendar) and Microsoft 365 (OneDrive, Outlook, Calendar, Excel)
  • Lead ingestion from multiple sources (Facebook Ads, Zapier, webhooks, manual entry)
  • A web-based administration panel for configuration and monitoring
